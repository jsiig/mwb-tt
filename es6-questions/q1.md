# Question 1: Write the above [here - below] using ES6 conventions

## Original ES5
```js
var expense = {
  type : 'Business',
  amount: '$50'
  }
  // Statements
  var type = expense.type;
  var amount = expense.amount;
```

I'm using `const` by default, `let` only if we know it will be reassigned.

```js
const expense = {
  type: 'Business',
  amount: '$50'
}

const { type, amount } = expense
```
